## Overview

`S4` (Structured State Space Sequence model) is an LTI LRNN that uses a diagonal plus low-rank (DPLR) parameterization of the state matrix A. This enables efficient computation while maintaining expressiveness through low-rank corrections.

S4 uses FFT-based convolution for parallel training and supports autoregressive inference via the step method.

## Example Usage

### Basic Usage

```python
from lrnnx.models.lti import S4
import torch

# Create S4 model
model = S4(d_model=64, d_state=64, l_max=1024)

# Training: parallel mode
x = torch.randn(2, 1024, 64)  # (batch, length, features)
y, state = model(x)
print(y.shape)  # torch.Size([2, 1024, 64])
```

### Autoregressive Inference

```python
# Allocate cache for inference
cache = model.allocate_inference_cache(batch_size=2)

# Generate sequence token by token
outputs = []
for t in range(100):
    x_t = torch.randn(2, 64)  # (batch, features)
    y_t, cache = model.step(x_t, cache)
    outputs.append(y_t)

# Stack outputs
y_seq = torch.stack(outputs, dim=1)  # (batch, length, features)
```

### With Gating and Bottleneck (GSS-style)

```python
model = S4(
    d_model=256,
    d_state=64,
    l_max=2048,
    bottleneck=2,      # Reduce inner dimension by 2x
    gate=2,            # Multiplicative gating with 2x expansion
    channels=4,        # Use 4 independent SSM heads
    dropout=0.1
)
```

## Key Features

### DPLR Parameterization

S4 uses a Diagonal Plus Low-Rank (DPLR) parameterization:

```
A = Λ - PP^T
```

where:
- Λ is a diagonal matrix (learnable complex eigenvalues)
- P is a low-rank correction matrix (rank determined by `rank` parameter)

This enables:
- **Efficient computation** via the Woodbury identity
- **Expressiveness** through low-rank corrections
- **Stability** via careful initialization

### FFT-Based Convolution

For training, S4 computes the SSM convolution in the frequency domain using FFT:

1. Compute convolution kernel K of length L
2. Apply FFT to both kernel and input
3. Multiply in frequency domain
4. Apply inverse FFT

This reduces complexity from O(L²) to O(L log L).

### Conjugate Symmetry

To ensure real-valued outputs from complex SSMs, S4 enforces conjugate symmetry in the state space, effectively halving the state dimension internally.

## Notes

- S4 uses `discretization="no_discretization"` as it handles discretization internally
- The `rank` parameter controls the expressiveness of the DPLR parameterization
- For long sequences, set `l_max` appropriately to pre-compute kernels
- The model includes a learnable skip connection via the D parameter
